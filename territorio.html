Territorio

Dónde está la red: focos de personas con demencia bajo control

Cada punto es un centro de salud real. El calor muestra dónde la red pública tiene más personas mayores con demencia bajo control: en celeste/claro, casi nadie; en morado, los focos donde se concentra el seguimiento. Haz clic en cualquier punto para ver el centro (y su tasa de detección).

Mapa de calor: focos de la red

ver código 
ok <- all(vapply(c("leaflet","leaflet.extras"), requireNamespace, logical(1), quietly = TRUE))
if (is.null(g)) {
  cat("> Falta productos/centros_geo.csv — corre R/06b_centros_geo.R")
} else if (!ok) {
  cat("> Para el mapa instala: install.packages(c('leaflet','leaflet.extras'))")
} else {
  suppressPackageStartupMessages({library(leaflet); library(leaflet.extras)})
  cel_morado <- c("#d6ecf8","#a9cbe8","#9a91d2","#8a44ad","#b73bb1","#e85fc6")
  vmax <- as.numeric(quantile(g$bajo_control_65[g$bajo_control_65 > 0], 0.75, na.rm = TRUE))
  if (!is.finite(vmax) || vmax <= 0) vmax <- 10
  pal <- colorNumeric(cel_morado, domain = c(0, vmax), na.color = "#e6ebee")
  gpos <- g[bajo_control_65 > 0]
  popup <- sprintf(
    "<b>%s</b><br>%s · %s<br>Bajo control 65+: <b>%s</b><br>Detección: %s",
    ifelse(is.na(g$nombre),"(s/n)",g$nombre), ifelse(is.na(g$comuna),"",g$comuna),
    ifelse(is.na(g$tipo),"",g$tipo),
    formatC(g$bajo_control_65, format = "d", big.mark = "."),
    ifelse(is.na(g$deteccion), "s/i (no APS)", paste0(round(100*g$deteccion), "%")))
  leaflet(options = leafletOptions(minZoom = 3)) |>
    addProviderTiles(providers$CartoDB.Positron) |>
    addHeatmap(data = gpos, lng = ~lon, lat = ~lat, intensity = ~bajo_control_65,
               radius = 24, blur = 32, minOpacity = 0.25, max = vmax, gradient = cel_morado) |>
    addCircleMarkers(data = g, lng = ~lon, lat = ~lat, radius = 3, stroke = TRUE,
                     weight = 0.4, color = "#5b6770", fillColor = ~pal(bajo_control_65),
                     fillOpacity = 0.85, group = "Centros (clic)", popup = popup) |>
    addLegend(data = g, pal = pal, values = ~bajo_control_65, position = "bottomright",
              title = "Bajo control 65+", opacity = 1) |>
    addLayersControl(overlayGroups = "Centros (clic)",
                     options = layersControlOptions(collapsed = FALSE)) |>
    setView(lng = -72, lat = -39, zoom = 5)
}

Calor = personas con demencia 65+ bajo control por centro (claro = pocas, morado = muchas). Haz clic en un punto para el detalle del centro.

El calor refleja volumen (cuántas personas bajo control), así que naturalmente brilla donde hay más población y más centros. Para ver qué tan bien detecta cada centro —no solo cuántos— usa el clic (muestra la tasa de detección) o la tabla de abajo. La tasa por centro es un proxy: limpia para atención primaria, “s/i” para hospitales.

Lo rural carga doble desventaja: mayor demencia esperada y menor registro. Reforzar la detección debería priorizar lo rural y extender los dispositivos del Plan Nacional.

Explorar los centros

ver código 
if (!is.null(g) && requireNamespace("DT", quietly = TRUE)) {
  tt <- g[order(-bajo_control_65)][, .(
    Comuna = comuna, Centro = nombre, Tipo = tipo,
    `Bajo control 65+` = bajo_control_65, Inscritos = inscritos_60,
    `Detección %` = ifelse(is.na(deteccion), NA, round(100*deteccion,1)))]
  DT::datatable(tt, rownames = FALSE, filter = "top",
                options = list(pageLength = 12, order = list(list(3, "desc"))))
} else {
  cat("> Para la tabla instala: install.packages('DT')")
}
