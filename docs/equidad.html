5 · Equidad

¿La brecha es más profunda para algunos grupos?

Miramos dos ejes independientes: si la brecha es peor para las personas más pobres, y para los pueblos originarios y migrantes. La inequidad más clara y robusta es la territorial (rural); los otros ejes matizan.

Eje pobreza (tramo A de FONASA = indigencia)

ver código 
if (!is.null(pob)) {
  pt <- pob[, .(`Comunas (de - a + pobres)` = cuartil_pobreza,
                `Indigencia media %` = tramo_a_medio,
                `Tasa detección %` = round(100*tasa_deteccion,1),
                `Brecha %` = round(100*brecha,1))]
  knitr::kable(pt, align = "lrrr")
}
Comunas (de - a + pobres) Indigencia media % Tasa detección % Brecha %
Q1 menos pobre 11.0 6.4 93.6
Q2 16.5 7.1 92.9
Q3 19.5 6.3 93.7
Q4 más pobre 26.7 4.7 95.3

No hay una línea recta entre pobreza y detección (la correlación es prácticamente cero). Pero el cuartil más pobre detecta claramente peor (4,7 % vs 6-7 %). Como esas comunas tienden a ser rurales, esto es probablemente el mismo efecto rural —no un efecto independiente de la pobreza.

Eje pueblos originarios y migrantes

ver código 
if (!is.null(ori)) {
  ot <- ori[, .(Zona = zona, `Bajo control` = bajo_control,
                `% pueblos originarios` = pct_pueblos_orig, `% migrantes` = pct_migrantes)]
  knitr::kable(ot, align = "lrrr", format.args = list(big.mark = "."))
}
Zona Bajo control % pueblos originarios % migrantes
PAIS 14.514 2.42 0.58
mixto 2.065 2.28 0.05
rural 1.710 6.67 0.12
urbano 3.776 1.59 0.93

Entre las personas con demencia bajo control, 2,4 % son de pueblos originarios a nivel país, con patrón territorial: 6,7 % en zonas rurales (donde vive más población originaria) vs 1,6 % urbano. Los migrantes son apenas 0,6 %.

El 2,4 % nacional parece bajo frente al peso poblacional de los pueblos originarios (~13 %), lo que sugiere posible subdetección — pero es una hipótesis: para confirmarla hace falta el denominador poblacional comunal (Censo 2024). Es la línea de trabajo a seguir en el eje de equidad étnica.

Las columnas de pueblos originarios y migrantes son marginales independientes: no se cruzan entre sí ni con edad/sexo, y la autoidentificación está subregistrada. El eje migrante es poco informativo para demencia (la migración es joven; la demencia, de 65+).

Datos REM (DEIS-MINSAL), preliminares 2025. Denominadores INE + FONASA. Proyecto reproducible.
